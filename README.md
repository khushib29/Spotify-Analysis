# Spotify-Analysis
This project investigates the factors influencing song popularity on Spotify using a dataset of 1,686 high-performing tracks. The study applies statistical analysis, hypothesis testing, and predictive modeling techniques to identify how various audio characteristics contribute to a song's success.

The analysis explores relationships between Spotify popularity scores and key musical attributes such as danceability, energy, loudness, acousticness, valence, and tempo. Additionally, differences in popularity across genres and subgenres are examined to uncover meaningful patterns in listener preferences.
